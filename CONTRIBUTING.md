### **Improving Documentation**
1. Fork the repository
2. Create a feature branch (`git checkout -b docs/improve-webhook-guide`)
3. Make your changes
4. Submit a Pull Request

### **Adding Examples**
We welcome code examples in any language! Add them to `docs/examples/`.

### **Updating OpenAPI Spec**
1. Edit `openapi/salesos-api.yaml`
2. Validate: `npx swagger-cli validate openapi/salesos-api.yaml`
3. Submit PR with description of changes
